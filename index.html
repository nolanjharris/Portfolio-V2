GNOMAD- A Visual Travel Blog

Developed August 2019

GNOMAD - A Visual Travel Blog

This Full-Stack Application allows the user to log all countries that they have traveled to. It will visualize the countries on an Interactive World Map in the color the user chooses. It also has a social aspect of adding friends and the functionality of toggling the map to show/hide countries that friends have traveled to. This site was made with React, SASS, Express, PostgreSQL, Leaflet.js(a JavaScript Mapping Library), and Cloudinary for cloud-based image storage.

Sudanku

Developed July 2019

Sudanku

Sudanku is a Web-App that allows the user to choose a Sudoku puzzle that they would like to solve. If the user gets stuck there is an option to get 'help' and it will fill in one of the squares with the right number. I wrote an algorithm to solve the board and limited it to one number at a time upon button press. There is also, of course, an algorithm that checks the solution at the end to verify that its a valid solution. Sudanku was made with React, Node and Express

Loopy

A simple Step Sequencer to make drum beats that will play on a continuous loop. It allows the user to change the tempo and set the amount of notes on the sequencer. Loopy was made with JQuery, and Tone.js (a javascript library for easy manipulation of the Web Audio API).

The ReachBack - Everything Disc Golf

With Disc Golf being one of my passions, I am very excited about this project. This is the main reason I feel comfortable with back-end concepts like databases, user authentication and RESTful routing. I used Node.js with Express, MongoDB, Passport, and several other packages to create The ReachBack. Users scores are logged into a database for later reference and the 'Stats' page shows a leaderboard for all users. Also areas to read about different courses and watch full disc golf videos.
